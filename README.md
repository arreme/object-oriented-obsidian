# Object Oriented Obsidian (OOO)
A minimalistic plugin to have object oriented notes in Obsidian. Setup target folders and the properties you want to validate. Press a button and done!
# How to setup
## Config
Very easy! Define target folder, properties to validate, and if you want to create objects with it
